List jobs posted by any target company

Given a target company, our Job Listings Endpoint will return an exhaustive list of all jobs posted by this company on LinkedIn. 

                
curl -X GET -H 'Accept: */*' -H 'Accept-Encoding: gzip, deflate' -H 'Authorization: Bearer <API_KEY>' -H 'Connection: keep-alive' 'https://nubela.co/proxycurl/api/v2/linkedin?url=https%3A%2F%2Fwww.linkedin.com%2Fin%2Ftimferriss
                
{
  "id": 110,
  "language": "Python",
  "price": 1900,
}
