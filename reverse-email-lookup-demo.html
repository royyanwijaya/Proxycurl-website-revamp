Automate Reverse Email Lookup with API

Given any email address, identify the corresponding social media profile and get rich profile data back.

Input
bill.gates@gatesfoundation.org

curl \
  -G \
  -H "Authorization: Bearer ${YOUR_API_KEY}" \
  'https://nubela.co/proxycurl/api/linkedin/profile/resolve/email' \
  --data-urlencode 'lookup_depth=deep' \
  --data-urlencode 'email=bill.gates@gatesfoundation.org' \
  --data-urlencode 'enrich_profile=enrich'
