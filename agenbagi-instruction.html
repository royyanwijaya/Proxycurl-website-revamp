How to install Chrome extension

Step by step to install a Chrome extension properly

Step 1: Invite friend with Chrome browser

Find with your friend if he/she uses a computer often (at least once every 2 days), and that it has Chrome browser

Step 2: Install ZIP file

Guide friend to install this ZIP file

Step 3: Unzip file

Unzip file in a safe directory

Step 4: Go to Chrome extension

Go to chrome://extensions

Step 5: Enable developer mode

Enable developer mode

Step 6: Load unpacked extension

Click on Load unpacked extension

Step 7: Choose directory

Choose directory

Final step: Connect user with your account

Get user to visit this link to connect his extension to your account
